Para correr el proyecto debes

1- Instalar las dependencias

npm install

////>>>desde aca

////>>>>> hasta aca

3- corre el siguiente comando para iniciar el proyecto (asegurate de tener configurada tu pc para correr react native con expo)

    npx expo start
